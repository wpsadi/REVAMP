Text Formatting Tags

  1. <strong> - Strong text
    • Use Case : used to bold text
    • Best Practice : prefered over bold as it provides extra importance to text
    • Using tag : This is a bolted text
  2. <em> - Emphasis text
    • Use Case : used to italize text
    • Best Practice : A screen reader will pronounce the words with an emphasis, using verbal stress.
    • Using tag : This is a emphasied text
  3. <del> - Delete text
    • Use Case : crosses/strikes through the word
    • Best Practice : This can be used when rendering "track changes" or source code diff information
    • Using tag : This is a deleted text
  4. <sup> - SuperScript text
    • Use Case : defines superscript text
    • Best Practice : Superscript text can be used for footnotes
    • Using tag : This is a superscripted text
  5. <sub> - SubScript text
    • Use Case : defines subscripted text
    • Best Practice : Subscript text can be used for chemical formulas
    • Using tag : This is a subscripted text
