HTML TAGS

  1. Anchor tag (a) : The anchor tag in HTML is used to create a hyperlink on the webpage.

  2. Body tag (body) :The body tag in HTML is used to define the main content present inside an HTML page.

  3. Break tag (br) :The break tag inserts a single carriage return or breaks in the document. This element has no end tag.

  4. Comment tag :The comment tag is used to insert comments in the HTML code.

  5. Division tag (div) :The div tag is used in HTML to make divisions of content in the web page

  6. Footer tag (footer) :The footer tag in HTML is used to define a footer of HTML document. This section contains the footer information.
